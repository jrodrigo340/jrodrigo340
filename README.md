### 🔥 Em destaque — Ecossistema **Já É** (Java / Spring Boot)

> Plataforma SaaS multi-tenant de gestão de negócios: agendamento, gestão de clientes e equipe, controle financeiro e um sistema de gamificação/níveis. Construída em dupla, com Spring Modulith e arquitetura hexagonal.

| Módulo | Descrição | Stack |
|---|---|---|
| **nf-file-normalizer** 🔒 | Serviço standalone de normalização de arquivos (parsing XML via DOM+XPath, agregação de erros com `sealed interface`, Strategy pattern) | Java 25, Spring Boot 4 |
| **agendamento** 🔒 | Módulo de agendamento de horários e serviços | Java, Spring Boot |
| **payment** 🔒 | Módulo de cobrança e assinaturas | Java, Spring Boot |
| **plataforma-scaffold** 🔒 | Base arquitetural compartilhada da plataforma | Java, Spring Modulith |

*(módulos marcados como 🔒 são repositórios privados — parte de um produto comercial em desenvolvimento)*
